# 很少出现过的场景题

- **超大表分流与迁移**：在 SQL 层把一个超大表的数据通过中间表同步到另一张表，如何设计任务拆分、索引/批量写入策略以降低锁表与流量峰值。来源：场景题/业务题/index.md。
- **动态审批流鲁棒性**：跨节点/跨角色的多级审批，支持退回、加签、转办；后端信息缺失或接口失败时的状态恢复与视图还原；离线场景（高铁审批）如何保留草稿。来源：场景题/业务题/index.md。
- **埋点追溯与审计**：审批失败后运营需要看到用户点击/查看过哪些页面，要求前端埋点链路串联。来源：场景题/业务题/index.md。
- **SSO 子域联登异常**：子域 A 登录后访问子域 B 是否需要重定向，Cookie 域如何设置，异常如何排查。来源：场景题/计算机网络/index.md。
- **终端差异排查**：微信端文件无法预览而浏览器正常，如何分层定位（客户端权限、文件格式、网关鉴权、CDN 缓存等）。来源：场景题/业务题/index.md。
- **智力博弈题**：100 个球，两人轮流取 1-2 个，如何构造必胜策略并证明。来源：场景题/智力题/index.md。
