### 前端基础与网络

1.  Service Worker 和 Web Worker 的区别和应用场景分别是什么？
2.  常用的前端缓存方案有哪些？它们各自的存储空间限制是多少，如果超出限制会发生什么？
    -   这个问题是从 Service Worker 的缓存策略延伸出来的。
    -   提到了 `localStorage` (大约 4MB) 和 `IndexedDB`。
3.  如何在本地开发（dev）环境中实现项目的热修复（hotfix）？
    -   面试者提到了可能与 `WebSocket` 相关。
4.  SSE (Server-Sent Events) 是如何开启的？它和 HTTP/2 的 Server Push 有什么区别？

### React 与性能优化

1.  在 React 中，如何定位到耗时的 Diff 操作？
    -   面试者提到了 `performance` 工具，以及如何通过它发现底层的耗时任务。
2.  虚拟滚动的实现原理是什么？如果是用 Canvas 布局，核心思路是怎样的？
    -   核心在于组件的复用。
3.  当你完成一项性能优化后，如何量化优化的效果？用哪些具体指标来衡量？
