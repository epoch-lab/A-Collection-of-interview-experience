
题目：假设你正在爬一个 n 阶的楼梯。你每次可以向上爬 1 阶、2 阶 或者 4 阶。请问，爬到楼顶（第 n 阶）一共有多少种不同的方法？（注意：不同的攀爬顺序被视为不同的方法。例如，1+2 和 2+1 是两种不同的方法。）


```js

const cache = new Map();
const func1 = (n) => {
    //  特殊处处理1到4，计算他们的方案数。
    if (n === 1) return 1;
    if (n === 2) return 2;
    if (n === 3) return 4;
    if (n === 4) return 8;
    
    // cache缓存相关的值
    if (cache.has(n)) {
        return cache.get(n);
    }
    const result = func1(n - 1) + func1(n - 2) + func1(n - 4);
    cache.set(n, result);

    return result;

};

```
