# Agent 实习面试总结贴

主播最近面试了多家公司的 Agent 实习岗位（深信服、Anker、MiniMax、万物云、吉比特），整体感受是：不再考“会不会用大模型”，而是考“能不能把 Agent 系统跑在真实工程里”。下面按模块整理高频问题。

## 总体印象
- 更关注工程落地与系统能力，而非简单调用模型
- 高频考点集中在上下文、工具调用、异步与多 Agent 协调等
